# Py-selenium


 > 公司没测试，隔一天就全量上线，写几个脚本，自动测试


## 需求说明

执行流程：

1. 新建模板。
2. 购买产品，续费产品
3. 修改产品详情


## 流程说明

### 新建模板
1. 登录网站
2. 新建模板
3. 元素确认：确认列表和新建内容符合

### 购买产品，续费产品
1. 登录网站
2. 选择产品
3. 加入购物车并结算
4. 查看产品列表
5. 核算新产品的时间。
6. 续费产品
7. 元素确认：核算新产品续费后的时间。


### 修改产品详情
1. 登录网站
2. 跳转到管理
3. 进入修改流程
	1. 修改信息
	2. 修改XX记录
	3. 点击保护按钮。

### 退费
1. 整体测试完成
2. 调用退费接口（or web登录）
3. 录入产品信息
4. 特殊退费-普通退费
5. 退费完成
